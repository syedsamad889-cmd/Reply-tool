# Sticker Mule — Reply Tool Clone (Mini)

A lightweight customer support reply tool built with React + TypeScript, inspired by Sticker Mule's internal tooling stack.

## Tech Stack
- React + TypeScript (Frontend)
- GraphQL (API layer simulation)
- PostgreSQL-compatible SQL schema
- Tailwind CSS (Styling)

## Features
- View and manage customer support tickets
- Reply to tickets with status updates
- Filter tickets by status (Open / Resolved / Pending)
- Clean, responsive UI

## How to Run
```bash
npm install
npm start
```

## Database Schema
See `schema.sql` for the PostgreSQL schema used for ticket management.
